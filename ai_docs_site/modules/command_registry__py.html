command_registry

Модуль реализует реестр команд для бота, формируя список доступных команд с их описаниями, обработчиками и флагами отображения в меню.
Список команд генерируется на основе конфигурации приложения, в частности, с учётом доступности MTProto-функциональности.
Каждая команда представлена словарём с ключами: имя, описание, обработчик и признак видимости в меню.

Ключевые структуры данных
Dict[str, object] — Словарь, описывающий команду с полями: name (имя команды), desc (описание), handler (функция-обработчик), menu (флаг отображения в меню)

build_command_registry(bot_app) -> List[Dict[str, object]]
Формирует список зарегистрированных команд бота с учётом конфигурации
Аргументы
bot_app — Экземпляр приложения бота, содержащий обработчики и конфигурацию
Возвращает
Список словарей, каждый из которых описывает доступную команду
Исключения
Отсутствуют
