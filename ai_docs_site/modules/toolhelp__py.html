    toolhelp

    Модуль предоставляет инструменты для управления справочной информацией о функциях (инструментах) в формате JSON. Данные хранятся в виде словаря, где ключом является имя инструмента, а значением — объект с описанием и временем обновления. Поддерживаются операции загрузки, сохранения, обновления и получения справки по инструменту.

    Ключевые структуры данных
    ToolHelpEntry — запись справки об инструменте, включающая имя инструмента, содержание справки и время последнего обновления

    load_toolhelp(path: str) -> Dict[str, ToolHelpEntry]
    Загружает справочные данные из JSON-файла по указанному пути
    Аргументы
    path — путь к файлу с данными в формате JSON
    Возвращает
    Словарь, где ключ — имя инструмента, значение — объект ToolHelpEntry
    Исключения
    Может вызвать исключения, связанные с чтением файла или парсингом JSON

    save_toolhelp(path: str, data: Dict[str, ToolHelpEntry]) -> None
    Сохраняет справочные данные в JSON-файл по указанному пути
    Аргументы
    path — путь к файлу для сохранения
    data — словарь с данными для сохранения
    Исключения
    Может вызвать исключения, связанные с записью файла

    update_toolhelp(path: str, tool: str, content: str) -> None
    Обновляет или добавляет справку для указанного инструмента с фиксацией времени обновления
    Аргументы
    path — путь к файлу хранения данных
    tool — имя инструмента
    content — текст справочной информации
    Исключения
    Может вызвать исключения при чтении или записи файла

    get_toolhelp(path: str, tool: str) -> Optional[ToolHelpEntry]
    Возвращает справочную запись для указанного инструмента, если она существует
    Аргументы
    path — путь к файлу с данными
    tool — имя запрашиваемого инструмента
    Возвращает
    Объект ToolHelpEntry, если найден, иначе None
    Исключения
    Может вызвать исключения при чтении файла
