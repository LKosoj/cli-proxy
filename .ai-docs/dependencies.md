# Зависимости

Проект использует следующие Python-пакеты, перечисленные в `requirements.txt`:

- **`python-telegram-bot`** — основа для реализации Telegram-бота, обеспечивает работу с API Telegram, обработку сообщений, колбэков и команд.
- **`PyYAML`** — парсинг и запись конфигурационного файла `config.yaml` с поддержкой сложных структур и переменных окружения.
- **`pexpect`** — управление интерактивными CLI-процессами (сессиями), ввод/вывод, обнаружение приглашений и тик-токенов.
- **`ansi2html`** — преобразование ANSI-цветов в HTML для отображения форматированного вывода в Telegram.
- **`requests`** — выполнение HTTP-запросов, включая рендеринг Mermaid-диаграмм через внешний сервис.
- **`pytest`** — модульное тестирование логики бота и утилит.
- **`telethon`** — альтернативная библиотека для работы с Telegram API через MTProto (опционально, требуется при включённой интеграции `mtproto`).
- **`markdown-it-py`** и **`mdit-py-plugins`** — безопасная конвертация Markdown в HTML с поддержкой задач, таблиц и разрывов строк.
- **`linkify-it-py`** — автоматическое распознавание и оборачивание URL в ссылки при обработке текста.

Все версии пакетов зафиксированы для обеспечения воспроизводимости окружения. Опциональные зависимости (например, `telethon`) используются только при включённых соответствующих функциях в конфигурации.
