# bot_logging

Модуль настройки централизованного логирования с поддержкой ротации файлов и специализированных обработчиков для разных уровней и источников логов.  
Конфигурирует три файла логов: основной (INFO и выше), ошибок (ERROR и выше) и агентов (только сообщения от логгеров с префиксом `agent.`).  
Также перехватывает необработанные исключения в главном потоке и в потоках, записывая их в лог.

Ключевые структуры данных
None — Модуль не определяет пользовательских типов или структур данных.

setup_logging(log_path: str) -> None
Настраивает систему логирования с ротацией по времени и разделением по уровням и источникам.
Аргументы
log_path — Путь к основному файлу лога; производные пути для ошибок и агентов генерируются автоматически.
Возвращает
None
Исключения
None — Функция не выбрасывает исключения, внутренние ошибки подавляются.
