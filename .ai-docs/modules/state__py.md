# state

```markdown
Модуль для управления состоянием сессий и активных сессий в файловой системе.
Предоставляет функции для загрузки, сохранения, обновления и удаления состояний сессий,
а также управления активной сессией и дополнительными сессионными данными.
Состояние хранится в JSON-файле, путь к которому передаётся в каждом вызове.

Ключевые структуры данных
SessionState — состояние сохранённой сессии, включая инструмент, рабочую директорию, токен возобновления, сводку и метаданные.
ActiveState — состояние активной сессии, включая инструмент, рабочую директорию, временну́ю метку и идентификатор сессии.

class SessionState
Состояние сохранённой сессии.
Поля
tool — используемый инструмент
workdir — рабочая директория
resume_token — токен для возобновления сессии (может быть None)
summary — текстовая сводка сессии (может быть None)
updated_at — временная метка последнего обновления (в формате timestamp)
name — опциональное имя сессии

---
class ActiveState
Состояние активной сессии.
Поля
tool — используемый инструмент
workdir — рабочая директория
updated_at — временная метка последнего обновления (в формате timestamp)
session_id — опциональный идентификатор сессии

---
def _load_raw(path: str) -> Dict[str, Any]
Загружает JSON-данные из файла или возвращает пустой словарь при ошибке или отсутствии файла.
Аргументы
path — путь к файлу состояния
Возвращает
Словарь с данными из JSON-файла или пустой словарь при ошибке
Исключения
Любые исключения логируются, но не пробрасываются дальше

---
def _save_raw(path: str, raw: Dict[str, Any]) -> None
Сохраняет словарь в файл в формате JSON с отступами.
Аргументы
path — путь к файлу состояния
raw — данные для сохранения в формате словаря
Исключения
Любые исключения логируются, но не пробрасываются дальше

---
def load_state(path: str) -> Dict[str, SessionState]
Загружает все сохранённые состояния сессий из файла.
Аргументы
path — путь к файлу состояния
Возвращает
Словарь с загруженными состояниями сессий

---
def save_state(path: str, data: Dict[str, SessionState]) -> None
Сохраняет состояния сессий в файл, дополняя существующие данные.
Аргументы
path — путь к файлу состояния
data — словарь состояний сессий для сохранения

---
def delete_state(path: str, tool: str, workdir: str) -> None
Удаляет состояние сессии по инструменту и рабочей директории.
Аргументы
path — путь к файлу состояния
tool — название инструмента
workdir — путь к рабочей директории

---
def make_key(tool: str, workdir: str) -> str
Формирует строковый ключ для идентификации сессии.
Аргументы
tool — название инструмента
workdir — путь к рабочей директории
Возвращает
Строковый ключ в формате "tool::workdir"

---
def update_state(path: str, tool: str, workdir: str, resume_token: Optional[str], summary: Optional[str], name: Optional[str] = None) -> None
Обновляет или создаёт состояние сессии с текущей временной меткой.
Аргументы
path — путь к файлу состояния
tool — название инструмента
workdir — путь к рабочей директории
resume_token — токен для возобновления сессии (может быть None)
summary — текстовая сводка сессии (может быть None)
name — опциональное имя сессии

---
def get_state(path: str, tool: str, workdir: str) -> Optional[SessionState]
Получает состояние сессии по инструменту и рабочей директории.
Аргументы
path — путь к файлу состояния
tool — название инструмента
workdir — путь к рабочей директории
Возвращает
Объект SessionState или None, если сессия не найдена

---
def load_active_state(path: str) -> Optional[ActiveState]
Загружает состояние активной сессии.
Аргументы
path — путь к файлу состояния
Возвращает
Объект ActiveState или None, если активная сессия не установлена

---
def set_active_state(path: str, tool: str, workdir: str, session_id: Optional[str] = None) -> None
Устанавливает активную сессию.
Аргументы
path — путь к файлу состояния
tool — название инструмента
workdir — путь к рабочей директории
session_id — опциональный идентификатор сессии

---
def clear_active_state(path: str) -> None
Очищает состояние активной сессии.
Аргументы
path — путь к файлу состояния

---
def load_sessions(path: str) -> Dict[str, Any]
Загружает дополнительные сессионные данные.
Аргументы
path — путь к файлу состояния
Возвращает
Словарь с дополнительными сессионными данными

---
def save_sessions(path: str, sessions: Dict[str, Any]) -> None
Сохраняет дополнительные сессионные данные.
Аргументы
path — путь к файлу состояния
sessions — словарь с дополнительными сессионными данными
```
