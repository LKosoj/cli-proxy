# tg_markdown

Модуль предоставляет функцию для безопасного преобразования текста в формат MarkdownV2, используемый Telegram.  
Основная цель — корректно экранировать специальные символы, чтобы текст отображался как ожидается в Telegram-сообщениях.  
При наличии установленной библиотеки `md2tgmd` используется её функция экранирования; в противном случае применяется консервативное экранирование всех специальных символов.

Ключевые структуры данных
_MDV2_SPECIALS — строка с перечнем специальных символов MarkdownV2, подлежащих экранированию
_MDV2_RE — скомпилированный регулярное выражение для поиска специальных символов MarkdownV2

to_markdown_v2(text: str) -> str
Преобразует текст, экранируя специальные символы MarkdownV2 для использования в Telegram.
Аргументы
text — текст для экранирования
Возвращает
Экранированная строка, безопасная для вставки в MarkdownV2
Исключения
Никакие исключения не обрабатываются явно; при ошибках используется fallback-режим
