# command_registry

Модуль предоставляет функцию для построения реестра команд бота, где каждая команда ассоциируется с обработчиком и метаданными.  
Реестр используется для маршрутизации пользовательских команд к соответствующим методам приложения.  
Каждая команда включает имя, описание, ссылку на обработчик и флаг видимости в меню.

Ключевые структуры данных
Dict[str, object] — Словарь, описывающий одну команду с полями: name (имя команды), desc (описание), handler (функция-обработчик), menu (флаг видимости в меню)
List[Dict[str, object]] — Список команд, возвращаемый функцией build_command_registry

build_command_registry(bot_app) -> List[Dict[str, object]]
Создаёт и возвращает список зарегистрированных команд бота с их обработчиками и метаданными.
Аргументы
bot_app — Экземпляр приложения бота, содержащий методы-обработчики команд
Возвращает
Список словарей, каждый из которых описывает одну доступную команду
Исключения
Не выбрасывает исключения
