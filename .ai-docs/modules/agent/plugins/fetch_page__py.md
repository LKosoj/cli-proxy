# agent/plugins/fetch_page

Модуль реализует инструмент для извлечения и парсинга веб-страниц по заданному URL. Инструмент возвращает содержимое страницы в виде чистого markdown-текста. Реализация делегирует непосредственную загрузку и обработку страницы вспомогательным функциям. Уровень риска использования инструмента — средний.

Ключевые структуры данных  
FetchPageTool — Плагин инструмента, предоставляющий функциональность извлечения и парсинга веб-страниц.

---
def get_spec() -> ToolSpec  
Возвращает спецификацию инструмента fetch_page.  
Аргументы  
Возвращает  
Объект `ToolSpec`, описывающий имя, описание, параметры и уровень риска инструмента.  
Исключения

---
async def execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]  
Асинхронно выполняет извлечение и парсинг веб-страницы по URL.  
Аргументы  
args — Словарь с входными аргументами, должен содержать ключ `url` с адресом страницы.  
ctx — Контекст выполнения, передаваемый плагину (не используется напрямую).  
Возвращает  
Словарь с результатом выполнения, содержащим распарсенный markdown-текст страницы.  
Исключения  
Возможны исключения, возникающие при сетевых запросах или обработке ответа (обрабатываются на уровне `helpers.fetch_page_impl`).
