# agent/plugins/website_content

Модуль предоставляет инструмент для извлечения и очистки основного текстового содержимого веб-страницы по заданному URL.  
Использует внешний сервис r.jina.ai для эффективного извлечения чистого текста без необходимости парсить HTML вручную.  
В случае недоступности Jina Reader выполняется fallback на прямое получение содержимого страницы.  
Поддерживает ограничение длины результата и интегрируется в асинхронную систему выполнения через `asyncio.to_thread`.

Ключевые структуры данных  
WebsiteContentTool — Реализация плагина инструмента для извлечения текста с веб-страниц

---
async def execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]  
Асинхронно выполняет извлечение и обработку текста с веб-страницы по URL.  
Аргументы  
args — Аргументы выполнения, включая `url` и опционально `max_chars`  
ctx — Контекст выполнения (не используется напрямую)  
Возвращает  
Словарь с результатом: `success`, `output` при успехе или `error` при ошибке  
Исключения  
Нет прямых исключений; все ошибки обрабатываются внутри метода

---
def _fetch_sync(self, url: str) -> str  
Синхронно извлекает текстовое содержимое страницы с использованием Jina Reader или прямого HTTP-запроса.  
Аргументы  
url — URL страницы для извлечения содержимого  
Возвращает  
Текстовое содержимое страницы или пустую строку  
Исключения  
RuntimeError — Если HTTP-запрос завершился с ошибкой статуса

---
def get_spec(self) -> ToolSpec  
Возвращает спецификацию инструмента, включая параметры, описание и настройки выполнения.  
Возвращает  
Объект `ToolSpec` с описанием возможностей инструмента

---
def get_source_name(self) -> str  
Возвращает имя источника инструмента для отображения.  
Возвращает  
Строку "Website Content"
