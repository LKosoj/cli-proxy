# agent/plugins/manage_tasks

Модуль предоставляет инструмент для управления списком задач в рамках сессии. Поддерживает операции добавления, обновления статуса, отображения и очистки завершённых задач. Используется для планирования и отслеживания многошаговых рабочих процессов агента.

Ключевые структуры данных  
ManageTasksTool — Плагин инструмента для управления задачами пользователя в рамках сессии

---
get_spec() -> ToolSpec  
Возвращает спецификацию инструмента для интеграции с системой планирования.  
Возвращает  
Объект ToolSpec с описанием имени, параметров и поведения инструмента

---
async execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]  
Выполняет действие с задачами: добавление, обновление, список или очистка.  
Аргументы  
args — Аргументы операции: действие и список задач с идентификаторами, содержанием и статусами  
ctx — Контекст выполнения, включая идентификатор сессии  
Возвращает  
Результат выполнения: статус успеха, выходные данные или сообщение об ошибке  
Исключения  
Не выбрасывает исключения напрямую

class ManageTasksTool
