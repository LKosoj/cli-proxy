# agent/plugins/search_files

```markdown
Модуль предоставляет инструмент для поиска файлов в проекте с использованием glob-шаблонов. Позволяет безопасно исследовать структуру проекта, блокируя попытки выхода за пределы рабочей директории. Исключает файлы и директории, связанные с системами контроля версий и зависимостями (например, `.git`, `node_modules`).

Ключевые структуры данных
ToolSpec — Описание интерфейса инструмента, включая параметры и поведение
ToolPlugin — Базовый класс для плагинов инструментов, определяющий контракт взаимодействия

---
def get_spec(self) -> ToolSpec
Возвращает спецификацию инструмента для поиска файлов.
Аргументы
Возвращает
Объект ToolSpec с описанием имени, параметров и назначения инструмента
Исключения

---
async def execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]
Выполняет поиск файлов по заданному glob-шаблону в контексте рабочей директории.
Аргументы
args — Аргументы выполнения, содержащие ключ `pattern` с glob-шаблоном
ctx — Контекст выполнения, содержащий `cwd` — текущую рабочую директорию
Возвращает
Словарь с результатом: `success` (успех/ошибка), `output` (список найденных файлов) или `error` (сообщение об ошибке)
Исключения
Любые исключения перехватываются и возвращаются как часть результата с `success=False`
```
