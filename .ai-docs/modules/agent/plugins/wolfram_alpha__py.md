# agent/plugins/wolfram_alpha

```markdown
Модуль реализует интеграцию с WolframAlpha через плагин инструмента, позволяя выполнять вычисления и получать ответы на запросы с использованием внешнего API. Запросы обрабатываются асинхронно, с поддержкой таймаутов и валидации входных данных. Для аутентификации требуется API-ключ, задаваемый через переменную окружения `WOLFRAM_APP_ID`.

Ключевые структуры данных
WolframAlphaTool — Плагин инструмента для выполнения запросов к WolframAlpha

---
async def execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]
Выполняет запрос к WolframAlpha с переданным текстом.
Аргументы
args — Аргументы запроса, включая строку `query` на английском языке
ctx — Контекст выполнения (не используется)
Возвращает
Словарь с результатом выполнения: успех, текст ответа или сообщение об ошибке
Исключения
Любые исключения перехватываются и преобразуются в словарь с ошибкой

---
def _call_sync(self, app_id: str, query: str) -> str
Синхронный вызов API WolframAlpha для получения ответа в виде текста.
Аргументы
app_id — Идентификатор приложения для доступа к API
query — Текст запроса
Возвращает
Текст ответа от WolframAlpha или сообщение об ошибке
Исключения
RuntimeError — При ошибке HTTP-запроса или недоступности сервиса

---
class WolframAlphaTool
Плагин инструмента для интеграции с WolframAlpha.
Методы
get_source_name() -> str — Возвращает имя источника: "WolframAlpha"
get_spec() -> ToolSpec — Возвращает спецификацию инструмента, включая параметры и таймаут
execute(args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any] — Асинхронно выполняет запрос к WolframAlpha
_call_sync(app_id: str, query: str) -> str — Синхронно вызывает API WolframAlpha и возвращает результат
```
