# agent/plugins/write_file

Модуль реализует инструмент для безопасной записи файлов в рамках рабочей области агента.  
Обеспечивает защиту от записи в чувствительные файлы, выхода за пределы рабочей директории и внедрения вредоносного кода.  
Использует вспомогательные функции из `helpers` для валидации путей и содержимого.

Ключевые структуры данных  
ToolSpec — Описание интерфейса инструмента, включая параметры и поведение

---
def get_spec() -> ToolSpec  
Возвращает спецификацию инструмента для записи файла.  
Возвращает  
Объект ToolSpec с описанием имени, параметров и поведения инструмента

---
async def execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]  
Выполняет запись содержимого в файл с предварительной проверкой безопасности.  
Аргументы  
args — Аргументы выполнения, включают 'path' и 'content'  
ctx — Контекст выполнения, включает рабочую директорию 'cwd'  
Возвращает  
Словарь с результатом операции: успех, ошибка или вывод  
Исключения  
Любое исключение при записи файла логируется и возвращается как ошибка выполнения
