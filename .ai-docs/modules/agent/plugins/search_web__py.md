# agent/plugins/search_web

Модуль реализует инструмент поиска в интернете как плагин для агента.  
Позволяет выполнять веб-поисковые запросы через внешний API и получать результаты в структурированном виде.  
Используется для получения актуальной информации: новости, цены, погода, определения и т.д.  
Интегрируется в систему инструментов агента через стандартный интерфейс `ToolPlugin`.

Ключевые структуры данных  
SearchWebTool — Плагин инструмента для выполнения веб-поиска по заданному запросу

class SearchWebTool  
Плагин инструмента веб-поиска, реализующий интерфейс `ToolPlugin`  
Методы  
def get_spec(self) — Возвращает спецификацию инструмента поиска в интернете  
Аргументы  
self — Экземпляр плагина  
Возвращает  
Объект `ToolSpec` с описанием инструмента, его параметрами и уровнем риска  
Исключения  
Не выбрасывает исключения

---
async def execute(self, args: dict, ctx: dict)  
Асинхронно выполняет веб-поисковый запрос и возвращает результат  
Аргументы  
self — Экземпляр плагина  
args — Аргументы выполнения, содержащие строку запроса `query`  
ctx — Контекст выполнения (не используется напрямую)  
Возвращает  
Словарь с результатами поиска, возвращёнными `search_web_impl`  
Исключения  
Может выбросить исключение при сетевой ошибке или недоступности поискового API
