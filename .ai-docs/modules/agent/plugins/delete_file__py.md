# agent/plugins/delete_file

Модуль реализует инструмент удаления файлов в рамках рабочей директории.  
Он обеспечивает безопасное удаление файлов с проверкой доступа и расположения.  
Инструмент интегрируется в систему плагинов через базовый класс `ToolPlugin`.

Ключевые структуры данных  
DeleteFileTool — Плагин инструмента для безопасного удаления файлов в рабочей директории

---
class DeleteFileTool  
Плагин инструмента для безопасного удаления файлов в рабочей директории  
Методы  
def get_spec() -> ToolSpec — Возвращает спецификацию инструмента удаления файла  
async def execute(args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any] — Выполняет удаление файла по указанному пути с проверкой безопасности

---
def get_spec() -> ToolSpec  
Возвращает спецификацию инструмента удаления файла.  
Возвращает  
Объект ToolSpec с описанием инструмента, его параметрами и требованиями

---
async def execute(args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]  
Выполняет удаление файла по указанному пути с проверкой безопасности.  
Аргументы  
args — Аргументы выполнения, содержащие путь к файлу  
ctx — Контекст выполнения, включая текущую рабочую директорию  
Возвращает  
Словарь с результатом операции: успех, сообщение об ошибке или вывод  
Исключения  
Любые исключения логируются и возвращаются как ошибка выполнения
