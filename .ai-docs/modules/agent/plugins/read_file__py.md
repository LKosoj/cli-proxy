# agent/plugins/read_file

Модуль реализует инструмент для безопасного чтения содержимого файлов в рамках рабочей области. Он проверяет права доступа, исключает чтение чувствительных и внешних файлов, поддерживает чтение по частям (смещение и лимит строк). Интегрируется в систему плагинов инструментов агента через базовый класс `ToolPlugin`.

Ключевые структуры данных  
ReadFileTool — Плагин инструмента для чтения файлов с проверкой безопасности и поддержкой частичного чтения

---
def get_spec() -> ToolSpec  
Возвращает спецификацию инструмента для чтения файла.  
Возвращает  
Объект ToolSpec с именем, описанием и схемой параметров инструмента

---
async def execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]  
Выполняет операцию чтения файла с проверкой безопасности и поддержкой смещения/лимита строк.  
Аргументы  
args — Аргументы выполнения: путь к файлу, опционально смещение и лимит строк  
ctx — Контекст выполнения, включает текущую рабочую директорию (cwd)  
Возвращает  
Словарь с результатом: успех, содержимое файла или сообщение об ошибке  
Исключения  
Любые исключения перехватываются и возвращаются как ошибка выполнения
