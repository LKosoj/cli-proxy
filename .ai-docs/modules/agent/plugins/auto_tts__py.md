# agent/plugins/auto_tts

Модуль реализует инструмент синтеза речи (TTS) через API OpenAI, интегрируемый в систему плагинов инструментов агента. Плагин генерирует аудиофайл из текста с поддержкой различных голосов, форматов и моделей. Результатом выполнения является путь к сохранённому временному аудиофайлу. Требуется наличие API-ключа OpenAI, заданного через переменную окружения или конфигурацию.

Ключевые структуры данных  
AutoTTSTool — Плагин инструмента для генерации речи с использованием OpenAI TTS

---
async def execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]  
Выполняет синтез речи по заданному тексту и параметрам, сохраняя результат в временный файл.  
Аргументы  
args — Аргументы инструмента: текст, голос, модель, формат и инструкции для озвучки  
ctx — Контекст выполнения (не используется напрямую)  
Возвращает  
Словарь с результатом: успех, путь к файлу или сообщение об ошибке  
Исключения  
Любые исключения при вызове API обрабатываются и возвращаются как ошибка в результирующем словаре

---
def get_source_name(self) -> str  
Возвращает имя источника инструмента.  
Возвращает  
Строку "OpenAI TTS"

---
def get_spec(self) -> ToolSpec  
Возвращает спецификацию инструмента TTS.  
Возвращает  
Объект ToolSpec с описанием параметров, модели выполнения и таймаута

---
class AutoTTSTool  
Плагин инструмента для генерации речи с использованием OpenAI TTS  
Методы  
get_source_name() -> str — Возвращает имя источника инструмента  
get_spec() -> ToolSpec — Возвращает спецификацию инструмента  
execute(args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any] — Асинхронно генерирует аудио из текста и возвращает путь к файлу
