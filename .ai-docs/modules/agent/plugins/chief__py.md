# agent/plugins/chief

Модуль предоставляет инструмент для поиска рецептов через API Edamam на основе заданных ингредиентов или кулинарных запросов.  
Реализует интерфейс `ToolPlugin` и позволяет асинхронно выполнять запросы к внешнему сервису, возвращая отформатированный результат.  
Конфигурация API осуществляется через переменные окружения `EDAMAM_APP_ID` и `EDAMAM_APP_KEY`.

Ключевые структуры данных  
ChiefTool — Плагин инструмента для поиска рецептов через Edamam API

class ChiefTool  
Плагин для поиска рецептов по запросу или ингредиентам через Edamam API  
Методы  
def get_source_name() -> str — Возвращает имя источника инструмента  
def get_spec() -> ToolSpec — Возвращает спецификацию инструмента, включая параметры и ограничения  
async def execute(args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any] — Асинхронно выполняет поиск рецептов по заданным параметрам  
def _search_sync(app_id: str, app_key: str, query: str, count: int) -> str — Синхронно выполняет HTTP-запрос к Edamam API и возвращает отформатированный результат

---
def _search_sync(app_id: str, app_key: str, query: str, count: int) -> str  
Выполняет синхронный запрос к Edamam API для поиска рецептов и возвращает отформатированный текстовый результат  
Аргументы  
app_id — Идентификатор приложения Edamam  
app_key — Ключ API Edamam  
query — Поисковый запрос (ингредиенты, блюдо и т.п.)  
count — Количество возвращаемых рецептов (ограничено от 1 до 10)  
Возвращает  
Отформатированная строка с названием, источником и ссылкой на рецепты, либо сообщение об отсутствии результатов  
Исключения  
RuntimeError — При ошибке HTTP-запроса или ответа от API
