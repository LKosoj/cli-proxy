# agent/plugins/list_directory

```markdown
Модуль реализует инструмент для безопасного просмотра содержимого директории в рамках изолированной рабочей среды. Он проверяет путь на соответствие разрешённым областям, блокирует доступ к системным и приватным директориям, а также выполняет команду `ls -la` для получения детального списка файлов. Инструмент интегрируется в систему плагинов через базовый класс `ToolPlugin`.

Ключевые структуры данных
ListDirectoryTool — Плагин инструмента для безопасного вывода содержимого директории

---
def get_spec(self) -> ToolSpec
Возвращает спецификацию инструмента для использования в системе планирования.
Аргументы
Возвращает
Объект ToolSpec с именем, описанием и параметрами инструмента
Исключения

---
async def execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]
Выполняет команду вывода содержимого директории с проверкой безопасности пути.
Аргументы
args — Аргументы команды, опционально содержащие путь к директории  
ctx — Контекст выполнения, включает текущую рабочую директорию (cwd)
Возвращает
Словарь с результатом выполнения: успех/ошибка и соответствующие данные (вывод или сообщение об ошибке)
Исключения
Любые исключения логируются и возвращаются как ошибка выполнения
```
