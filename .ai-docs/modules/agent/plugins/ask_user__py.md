# agent/plugins/ask_user

Модуль реализует инструмент взаимодействия с пользователем через вопрос с выбором из предложенных вариантов.  
Используется в рамках системы планирования агента для получения подтверждения или выбора действия от пользователя.  
Вопрос отправляется через бота, ответ ожидается асинхронно с таймаутом 120 секунд.

Ключевые структуры данных  
ToolSpec — Описание интерфейса инструмента, включая имя, параметры и поведение  
ToolPlugin — Базовый класс для плагинов инструментов, обеспечивает интеграцию с системой исполнения

---
async def execute(self, args: Dict[str, Any], ctx: Dict[str, Any]) -> Dict[str, Any]  
Выполняет отправку вопроса пользователю и ожидает выбор ответа.  
Аргументы  
args — Аргументы выполнения: вопрос и список опций выбора  
ctx — Контекст выполнения, содержащий бота, чат, сессию и сервисы  
Возвращает  
Результат выполнения: успешный ответ с выбором пользователя или ошибка  
Исключения  
asyncio.TimeoutError — Если пользователь не ответил в течение 120 секунд
