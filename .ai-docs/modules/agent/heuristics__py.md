# agent/heuristics

Модуль предоставляет утилиты для обработки шагов плана и определения необходимости уточнения входного текста на основе конфигурации. Основные функции включают проверку текста на признаки вопросов и автоматическую нормализацию шагов, требующих уточнений. Использует конфигурацию приложения и контракт `PlanStep` для согласованной обработки.

Ключевые структуры данных  
PlanStep — Описание шага в плане, включающего вопрос и возможные ответы

bool needs_clarification(const std::string& text, const AppConfig& config)  
Проверяет, требует ли текст уточнения на основе конфигурации  
Аргументы  
text — текст для анализа  
config — конфигурация приложения  
Возвращает  
True, если текст содержит вопрос или ключевые слова уточнения  
Исключения  
Отсутствуют

---
void normalize_ask_step(PlanStep& step)  
Нормализует шаг плана, устанавливая значения по умолчанию для вопроса и вариантов ответа  
Аргументы  
step — изменяемый объект шага плана  
Возвращает  
Ничего  
Исключения  
Отсутствуют
