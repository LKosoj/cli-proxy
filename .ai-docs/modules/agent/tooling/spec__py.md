# agent/tooling/spec

Модуль предоставляет базовые структуры для описания и использования инструментов (tools) в системах, взаимодействующих с ИИ-моделями.  
Основной класс `ToolSpec` описывает спецификацию инструмента с поддержкой преобразования в форматы, совместимые с OpenAI и Google.  
Класс `ToolResponse` представляет стандартный формат ответа от выполненного инструмента.

Ключевые структуры данных  
ToolSpec — Описание спецификации инструмента, включая параметры, метаданные и настройки выполнения.  
ToolResponse — Структура ответа от выполненного инструмента, включающая статус, результат и дополнительные метаданные.

class ToolSpec  
Описывает спецификацию инструмента, используемого в ИИ-агентах.  
Поля  
name — Имя инструмента, уникальный идентификатор.  
description — Описание функциональности инструмента.  
parameters — Схема параметров инструмента в формате JSON Schema.  
timeout_ms — Максимальное время выполнения в миллисекундах (по умолчанию 120 000).  
risk_level — Уровень риска выполнения: "low", "medium", "high" (по умолчанию "low").  
requires_approval — Требуется ли ручное подтверждение перед выполнением (по умолчанию False).  
tags — Список тегов для категоризации инструмента (по умолчанию пустой список).  
parallelizable — Можно ли выполнять инструмент параллельно с другими (по умолчанию True).  
Методы  
to_openai_tool() — Преобразует спецификацию в формат, совместимый с OpenAI.  
to_google_tool() — Преобразует спецификацию в формат, совместимый с Google AI.

---
class ToolResponse  
Представляет результат выполнения инструмента.  
Поля  
success — Успешно ли выполнен инструмент.  
output — Результат выполнения в виде строки (если есть).  
error — Сообщение об ошибке (если выполнение завершилось неудачно).  
meta — Дополнительные метаданные выполнения (например, время, контекст).

---
def to_openai_tool(self) -> Dict[str, Any]  
Преобразует спецификацию инструмента в формат OpenAI.  
Аргументы  
Возвращает  
Словарь, совместимый с форматом инструментов OpenAI.  
Исключения  
Не выбрасывает исключения.

---
def to_google_tool(self) -> Dict[str, Any]  
Преобразует спецификацию инструмента в формат Google AI.  
Аргументы  
Возвращает  
Словарь, совместимый с форматом инструментов Google.  
Исключения  
Не выбрасывает исключения.
