# agent/tooling/loader

```markdown
Класс `PluginLoader` предназначен для динамической загрузки плагинов инструментов из указанной директории. Он сканирует файлы Python, исключая системные и базовые, и загружает модули, содержащие классы, наследующиеся от `ToolPlugin`. Каждый найденный валидный класс инстанцируется и возвращается в виде списка активных плагинов.

Ключевые структуры данных
ToolPlugin — базовый класс, от которого должны наследоваться все плагины инструментов

---
class PluginLoader
Загружает плагины инструментов из указанной директории
Поля
plugins_directory — путь к директории с плагинами
Методы
def __init__(self, plugins_directory: Path) -> None — инициализирует загрузчик с указанием директории плагинов
def load(self) -> List[ToolPlugin] — сканирует директорию и возвращает список загруженных и созданных экземпляров плагинов
def _load_module(self, path: Path) — загружает модуль Python из указанного файла и возвращает объект модуля или None при ошибке
```
