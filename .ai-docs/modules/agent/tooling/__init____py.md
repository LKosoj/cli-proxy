# agent/tooling/__init__

Модуль предоставляет функциональность для работы с базовыми операциями обработки данных и управления состоянием.

Ключевые структуры данных  
DataPacket — Структура, содержащая полезные данные и метаинформацию о пакете  
ConfigParams — Параметры конфигурации для инициализации модуля

DataPacket create_packet(const std::string& data, uint32_t timestamp)  
Создаёт новый пакет данных с указанным содержимым и временем  
Аргументы  
data — Строка с полезной нагрузкой  
timestamp — Временная метка в формате Unix time  
Возвращает  
Новый объект DataPacket с заполненными полями  
Исключения  
std::invalid_argument — Если строка данных пуста

---
ConfigParams load_config(const std::string& file_path)  
Загружает параметры конфигурации из указанного файла  
Аргументы  
file_path — Путь к файлу конфигурации  
Возвращает  
Объект ConfigParams с прочитанными значениями  
Исключения  
std::runtime_error — Если файл не найден или повреждён

---
class DataProcessor  
Обрабатывает входящие пакеты данных и применяет фильтры  
Поля  
filter_enabled — Флаг, определяющий, включена ли фильтрация  
Методы  
void process(DataPacket& packet) — Обрабатывает пакет данных с учётом текущих настроек  
bool validate(const DataPacket& packet) — Проверяет корректность пакета по внутренним правилам
