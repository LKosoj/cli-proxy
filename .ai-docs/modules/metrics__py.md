# metrics

```markdown
Класс для сбора и отображения эксплуатационной статистики приложения. Поддерживает счётчики событий, отслеживание времени последнего вывода и формирование читаемого отчёта. Все данные хранятся в памяти, сброс состояния происходит при инициализации.

Ключевые структуры данных
Metrics — Класс для сбора и вывода метрик выполнения приложения

class Metrics
Класс для сбора и отображения метрик выполнения приложения

Поля
_counters — Словарь счётчиков событий, где ключ — имя счётчика, значение — накопленное количество
_last_output_time — Время последнего вызова вывода (в секундах с эпохи)
_output_count — Общее количество операций вывода
_total_output_chars — Общее количество выведенных символов

Методы
Metrics.__init__(self) — Инициализирует новый экземпляр счётчика метрик с пустыми значениями
---
Metrics.inc(self, key, count=1) — Увеличивает счётчик с указанным ключом на заданное значение
Аргументы
key — Имя счётчика (строка)
count — Значение, на которое увеличивается счётчик (по умолчанию 1)
Возвращает
None
---
Metrics.observe_output(self, chars) — Фиксирует факт вывода указанного количества символов и увеличивает счётчик выводов
Аргументы
chars — Количество выведенных символов
Возвращает
None
---
Metrics.snapshot(self) — Формирует строковое представление текущих метрик
Возвращает
Строка с информацией о времени работы, счётчиках и времени последнего вывода
```
