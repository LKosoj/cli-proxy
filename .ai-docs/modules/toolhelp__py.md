# toolhelp

Модуль предоставляет простую систему хранения и управления справочной информацией для инструментов (tools) в формате JSON-файла.  
Данные сохраняются с меткой времени обновления, что позволяет отслеживать актуальность справки.  
Поддерживаются операции загрузки, сохранения, обновления и получения справки по имени инструмента.

Ключевые структуры данных  
ToolHelpEntry — запись справки для инструмента, содержащая имя инструмента, содержание справки и время последнего обновления

class ToolHelpEntry  
Запись справочной информации для одного инструмента  
Поля  
tool — имя инструмента  
content — текст справки  
updated_at — временная метка последнего обновления в формате Unix time

---
def load_toolhelp(path: str) -> Dict[str, ToolHelpEntry]  
Загружает справочные данные из JSON-файла по указанному пути  
Аргументы  
path — путь к файлу с данными справки  
Возвращает  
Словарь, где ключ — имя инструмента, значение — объект ToolHelpEntry

---
def save_toolhelp(path: str, data: Dict[str, ToolHelpEntry]) -> None  
Сохраняет справочные данные в JSON-файл  
Аргументы  
path — путь к файлу для сохранения  
data — словарь с данными справки для сохранения

---
def update_toolhelp(path: str, tool: str, content: str) -> None  
Обновляет или добавляет справку для указанного инструмента с текущей временной меткой  
Аргументы  
path — путь к файлу данных справки  
tool — имя инструмента  
content — текст справки

---
def get_toolhelp(path: str, tool: str) -> Optional[ToolHelpEntry]  
Получает справку для указанного инструмента  
Аргументы  
path — путь к файлу данных справки  
tool — имя инструмента  
Возвращает  
Объект ToolHelpEntry, если найден, иначе None
