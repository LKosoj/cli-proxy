# toolhelp.json

Файл содержит справочную информацию о доступных slash-командах для различных инструментов искусственного интеллекта, используемых в CLI-режиме. Каждая запись соответствует конкретному инструменту и включает его название, тип, описание доступных команд и время последнего обновления. Описание команд ориентировано на пользователей, взаимодействующих с интерфейсом через текстовый ввод. Предназначен для быстрого ознакомления с функциональными возможностями и навигацией в рамках соответствующего CLI-инструмента.

Секции и ключи

codex — секция с информацией о CLI-инструменте Codex, включая список доступных slash-команд и способ вызова справки<br>
claude — секция с описанием встроенных команд для инструмента Claude Code, включая основные команды и ссылку на справку<br>
gemini — секция, описывающая команды Gemini CLI, с примерами использования и альтернативными синонимами команд<br>
qwen — секция с информацией о командах Qwen Code, включая базовые управляющие и настроечные команды<br>
tool — идентификатор используемого инструмента ИИ в каждой секции<br>
content — текстовое описание доступных команд и способов получения справки для соответствующего инструмента<br>
updated_at — временная метка последнего обновления информации в секции, указанная в формате Unix time

Важные параметры

/help — универсальная команда для вызова справки по доступным командам в большинстве инструментов<br>
/quit или /exit — команды для завершения сеанса, присутствуют во всех описанных инструментах<br>
/compact — команда для сжатого отображения информации, доступна в нескольких инструментах<br>
/model — команда для управления моделью ИИ, указана в Codex как одна из основных
