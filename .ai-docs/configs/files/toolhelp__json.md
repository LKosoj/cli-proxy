# toolhelp.json

Файл содержит справочную информацию о доступных slash-командах для различных инструментов искусственного интеллекта, используемых в CLI-режиме. Каждая запись соответствует конкретному инструменту и включает его название, тип, описание с перечнем команд и временную метку обновления. Данные предназначены для быстрого ознакомления с функциональными возможностями каждого инструмента через встроенные команды. Информация унифицирована для сравнения и использования в интерактивной среде.

Секции и ключи

codex — секция с информацией о CLI-инструменте Codex, включая список доступных slash-команд и способ вызова справки<br>
claude — секция, описывающая встроенные команды для инструмента Claude Code, с указанием базовых команд и справки<br>
gemini — раздел, содержащий команды Gemini CLI, включая утилиты для чата, настройки, аутентификации и управления состоянием<br>
qwen — информация о командах в Qwen Code, включая базовые управляющие и настроечные команды<br>
tool — идентификатор используемого инструмента ИИ в соответствующей секции<br>
content — текстовое описание доступных команд и способов получения справки для каждого инструмента<br>
updated_at — временная метка последнего обновления информации в секции, указанная в формате Unix time

Важные параметры

/help — универсальная команда для вызова справки по доступным командам в большинстве инструментов<br>
/quit или /exit — команды для завершения сеанса, присутствуют во всех представленных инструментах<br>
/compact — команда для сжатия или компактного отображения контекста, доступна в нескольких инструментах<br>
/auth — команда для управления аутентификацией, присутствует в Gemini и Qwen<br>
/theme — команда для изменения темы интерфейса, доступна в Gemini и Qwen
